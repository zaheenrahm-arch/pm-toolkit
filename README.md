# pm-toolkit

10 AI-powered PM tools built to spec. Each one demonstrates a core product management skill through a working interface — PRD writing, sprint retros, OKR setting, user research synthesis, and more.

Built with the Anthropic API. Each tool runs in the browser with no backend required.

## Usage

Each tool runs entirely in the browser. You'll need an [Anthropic API key](https://console.anthropic.com) to use them — enter it in the key field at the top of any tool. Your key is never stored or sent anywhere other than directly to Anthropic.
